# [castle-game](https://tversteeg.itch.io/castle-game)
A free & open source 2D Lemmings-meets-tower defense destructible terrain game.

# Prerequisites

## Linux (Debian based)

    sudo apt install xorg-dev

## Windows & Mac

None

# Run

    cargo run --release
